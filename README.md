# Car-Rental-Management-System
# 🚗 Car Rental System (C++)

A console-based Car Rental Management System built using **C++**, implementing **Object-Oriented Programming (OOP)** concepts and **File Handling**.

---

## 📌 Features

* 🔐 Login System (Admin authentication)
* 🚘 View all available cars
* 📅 Rent a car with timestamp
* 🔁 Return car with automatic bill calculation
* 📂 Active rentals tracking (file-based)
* 📊 Rental history management
* 💾 Data stored using file handling

---

## 🛠️ Technologies Used

* C++
* OOP Concepts (Classes, Objects)
* File Handling (ifstream, ofstream)
* STL (Vector)
* Date & Time (chrono, ctime)

---

## 📁 Project Structure

* `main.cpp` → Main program file
* `active_rentals.txt` → Stores current rentals
* `rental_history.txt` → Stores completed rentals

---

## ▶️ How to Run

1. Open project in VS Code / CodeBlocks
2. Compile the code:

   ```
   g++ main.cpp -o car_rental
   ```
3. Run the program:

   ```
   ./car_rental
   ```
